# Vrouwengilde Goes

Moderne, statische website voor Vrouwengilde Goes. De site is gebouwd met Next.js, TypeScript en Tailwind CSS en is klaar voor Netlify.

## Lokaal starten

```powershell
npm.cmd install
npm.cmd run dev
```

Open daarna `http://localhost:3000`.

## Controles

```powershell
npm.cmd run lint
npm.cmd run typecheck
npm.cmd run build
```

## Netlify

Netlify gebruikt:

- Build command: `npm run build`
- Publish directory: `out`

## Nog in te vullen placeholders

- Actuele contributie en voorwaarden
- Naam van het goede doel
- Actuele datums en locaties van activiteiten
- Namen of rolverdeling van het bestuur
- Echte foto's voor de fotopagina
